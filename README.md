# Formulador Proyectos Llave en Mano

Esta carpeta es una copia de trabajo de la app original creada en Snack/Expo.
La idea es mejorarla aqui sin tocar la version que tienes guardada en Snack.

## Estado del proyecto

- La app ya es un proyecto Expo normal.
- No depende de Snack para funcionar.
- Se puede seguir editando en un entorno externo con seguridad.

## Archivos principales

- `App.js`: flujo principal de la app.
- `screens/`: pantallas del formulario, CTV, estudio, historial y login.
- `utils/`: calculos, formateadores y exportacion PDF.
- `assets/`: iconos e imagenes del proyecto.

## Scripts disponibles

- `npm start`
- `npm run web`
- `npm run android`
- `npm run ios`

## Recomendacion de trabajo

Usa esta copia como entorno seguro para mejoras. La version original en `Downloads`
queda intacta como respaldo.

## Observaciones

- El `slug` de Expo ya fue cambiado para que no quede con el identificador temporal de Snack.
- El proyecto todavia conserva algunos textos con caracteres mal codificados, por ejemplo tildes y simbolos especiales. Eso no impide migrarlo, pero conviene corregirlo en una siguiente pasada.
